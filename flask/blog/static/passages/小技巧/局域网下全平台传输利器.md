# Feem——全平台文件传输软件

![](https://img.iplaysoft.com/wp-content/uploads/2019/feem/feem.jpg!0x0.webp)

| 优点         | 缺点         |
| ------------ | ------------ |
| 全平台       | 仅局域网传输 |
| 易用、速度快 | 免费版有广告 |
| 免费版无限制 | 官网被墙     |

**特点：**

- 自动配对连接 / 不需注册登录帐号
- 局域网传输 / 速度飞快而且更安全
- WiFi 热点直连 (离线传输文件 / 无需路由器)
- 支持文字传输
- 全平台支持 / 跨设备双向互传
- WebShare 网页文件共享，无需安装客户端
